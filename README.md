# My 100 Days of Code

Afin de revoir les bases de certains langages et m'améliorer en développement web, j'ai décidé de commencer le défi #100DaysOfCode. Ce défi consiste à coder tous les jours pendant au moins une heure. Ensuite, il faut push son code et partager un résumé du travail de la journée. Personellement, je partage mon aventure sur [Instagram](https://instagram.com/ana_dev_100?igshid=YmMyMTA2M2Y=). N'hésitez pas à me rejoindre !

Je ferais également un résumé de mon travail de chaque jour dans ce fichier pour garder une trâce de ma progression.

## JOUR 1 : HTML
L'aventure commence.
Aujourd'hui je me suis concentrée sur les bases du HTML. En effet, j'ai pensé qu'il était préférable de revoir les bases avant de me lancer pleinement dans ce challenge. De plus, un petit rappel ne fait de mal à personne ! ;)
J'ai donc lu la documentation de [MDN](https://developer.mozilla.org/fr/docs/Learn/Getting_started_with_the_web/HTML_basics) pour me remettre en mémoire les utilisations des balises les plus utilisées à savoir les titres, les images ou encore les liens.
J'ai décidé de créer la base html d'une page "à propos de moi" sur laquelle je me baserais pour la suite de la semaine.
J'ai également installé l'application Mimo Code sur laquelle je m'entraînerais en parallèle au challenge.
